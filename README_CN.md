# 🦞🧪 龙虾蒸馏

**跨平台加密技能传授系统，用于 OpenClaw 智能体之间的技能传授。**

[English](README.md) | [中文](README_CN.md)

## 快速开始

**发送技能：**
```bash
bash share.sh <技能目录> "描述"
```

**接收技能：**
```bash
bash receive.sh <链接> <密码> <名称> tar
```

详细文档请阅读 [SKILL.md](SKILL.md)。

## 特性

- 🌐 **跨平台** — 支持任意 IM：Telegram、微信、Discord、Signal、邮件……
- 🔐 **AES-256 加密** — 一次一密，安全传输
- ⏰ **24小时自动销毁** — 文件到期自动删除
- 🤝 **人类参与审核** — 管理员通过复制粘贴完全掌控
- 🎯 **极致简单** — 1 条命令发送，5 行命令接收
- 📦 **零依赖** — 仅使用 openssl、curl、tar（系统自带）

## 为什么选龙虾蒸馏？

| 特性 | 龙虾蒸馏 🦞 | Google A2A | ClawHub Publish |
|---|---|---|---|
| 跨平台 | ✅ 任意 IM | ❌ 仅 API | ✅ Web |
| 人类审核 | ✅ 必须 | ❌ AI 直连 | ❌ 自动安装 |
| 私有技能 | ✅ 点对点 | ❌ 需 API 接入 | ❌ 仅公开 |
| 加密 | ✅ AES-256 | 取决于实现 | ❌ 明文 |
| 依赖 | 无（系统自带） | SDK + API | npm + 网络 |

## 安全说明

- **人类参与审核**：没有人类主动转发，什么都不会发送
- **AES-256-CBC + PBKDF2**：随机一次性密码加密
- **24小时自动销毁**：上传的文件到期自动删除
- **无 AI 直连**：与基于 API 的协议不同，智能体之间没有直接网络连接
- **仅清理临时文件**：脚本只删除自己在 `/tmp/` 下创建的临时文件
